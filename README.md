# Tym-TQ-main

Trang web da duoc nang cap thanh mot bo hoa de thuong gui den ban nu ten Han.

## Chay nhanh

1. Mo file `index.html` bang trinh duyet.
2. Neu muon dung mp3 that, dat file nhac vao `assets/audio/romantic.mp3`.
3. Bam "Mo hop qua 3D" roi bam "Gui bo hoa cho Han".

## Diem nhan

- Giao dien hien dai, mau sac ngot ngao va than thien mobile.
- Bo hoa dong gom nhieu lop canh hoa, la, ruy bang va bong do.
- Hieu ung canh hoa roi bang canvas + animation lien tuc.
- Hieu ung go chu loi chuc va tuong tac khi gui hoa.
- Nhac nen nhe bang Web Audio va nut bat/tat am thanh.
- Ho tro nhac mp3 that qua file rieng `assets/audio/romantic.mp3`.
- Hieu ung mo hop qua 3D truoc khi bo hoa xuat hien.
- Trang phu `thiep.html` la tam thiep ky ten rieng: Cường.
